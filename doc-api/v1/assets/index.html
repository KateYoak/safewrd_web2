Assets API

An Asset is a media representation in a specific format, served from a CDN. An Asset represents and belongs to either a published medium, a user upload or a shared medium clip, usually in the form of a thumbnail image or a transcoded video file. Assets are generated by the system as a result of a publisher publishing a video, or a user uploading a file or sharing a clip.

CDN URLs

All static assets are stored on and served from Rackspace CloudFiles. These files are found in the following containers:

clqs-media
http://4c3d9c3ea6cfb30ee0a2-bfaedbe6cb0b27294c586ccf226ead67.r47.cf2.rackcdn.com
This container serves published media videos, thumbnails and AMDB files;
clqs-clips
http://b8b6b019a2f1c2347c26-44bf840199f7a28a2faa8dad8b80bc51.r29.cf2.rackcdn.com
This container serves 1-minute clippings from published media files and their thumbnails;
clqs-uploads
http://b17e220c835537819990-29ca84743ba02df04baae069712c11c6.r86.cf2.rackcdn.com
This container serves user-uploaded videos and their thumbnails;
clqs-images
http://7a4cab3e755e366365a1-3cc5852ef26862693a0b30d031a0e546.r58.cf2.rackcdn.com
This container serves user-uploaded images, such as profile pictures or Kliq front images.
