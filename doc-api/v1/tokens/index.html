Create a new authorization token

POST /tokens

Input

token
Required string - The access_token that can be used at the social networks to make requests on behalf of the user.
secret
Optional string - The access_token_secret that is needed for some providers.
service
Required string - The name of the identity service this token is valid for; either google, twitter, facebook, yahoo or linkedin.
handle
Optional string - The unique ID of the user at the social network.
{
  "service": "twitter",
  "token": "46722818-tlQvHw6NrHuTrhJuyoeXeCE5n9X6RTj6EfEiCg71J"
}

Response

Status: 201 Created
Location: https://api.tranzmt.it/v1/tokens/C68CFA6E-BDB4-1014-9C14-ABF6C84F2656
