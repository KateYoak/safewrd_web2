Uploads API

User-generated videos may be uploaded to the Tranzmt platform with any of the file extensions .mp4, .m4v, .mpeg, .mpg, .3gp or .webm.

After uploading, several mobile-optimized video- and thumbnail formats of the video source will be transcoded and put on the CDN, and made available as nested Asset objects in the Upload resource.
