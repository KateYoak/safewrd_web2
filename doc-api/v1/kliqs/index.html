Create a kliq

POST /kliqs

Input

name
Required string - Name of the Kliq.
isEmergency
Optional integer - Set to 1 if the Kliq is a emergency Kliq.
image
Optional upload file handle - Currently not implemented.
contactsMap
Required list of objects - Contacts that should be added to the Kliq.
{
  "name": "Cool kliq",
  "isEmergency": 1,
  "contactsMap": [
    {
      "contactId": "1A9F1F2B-A297-1014-8D7F-A3F7C84F2656"
    }
  ]
}
