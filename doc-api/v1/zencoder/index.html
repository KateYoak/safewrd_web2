Systems API

Zencoder notification

POST /zencoded

This is the callback URL used by Zencoder to post encoding output results to. For published media, user uploads and shared clippings, the output is recorded as an Asset. In addition, for published media outputs the fingerprinting worker process is triggered.

Input

Zencoder.com output notification.

Response

Status: 200 OK
{
  "message": "Encoding output received"
}
